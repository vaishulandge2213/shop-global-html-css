## Features
- Wide range of categories including Fashion, Electronics, Toys, Jewellery, Cosmetics, and Grocery.
- Easy and secure shopping experience with smooth navigation and safe payments.
- Exclusive online deals, discounts, and special offers for customers.
- Fast delivery, reliable service, and customer-first support system.


## Future ideas
- Expand product categories with advanced filtering options.
- Introduce user accounts with wishlist and order history.
- Implement secure online payment and checkout system.
- Add AI-based product recommendations for better user experience.
